## :wrench: Setup

1. Clone this repository

```bash
$ git clone https://github.com/giangpham-cfg/Zoo-feeder-project.git
```

2. Go into the repository

```bash
$ cd Zoo-feeder-project
```

3. Install dependencies

```bash
$ npm install
```

4. Run the app

```bash
$ npm run dev
```
